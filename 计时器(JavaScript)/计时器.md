# 计时器(JavaScript)

## 大概

- 计时器功能，可以设置计时时间，并显示剩余时间。
- 计时器可以暂停、继续、重置。
- 计时器可以设置是否循环。

## 实现(主要讲JavaScript的实现)

### 1.setTimeout()函数

- setTimeout()函数可以设置一个定时器，在指定的时间后执行一个函数。
- 语法：setTimeout(function, time);
- function：要执行的函数。==(也可以是一段代码，用括号包起来)==
- time：定时器的执行时间，<font color='red'>_单位为毫秒_</font>。
