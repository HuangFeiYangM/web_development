# 解构赋值

## 什么是解构赋值？

解构赋值是一种 JavaScript 表达式，它允许将数组或对象中的值，按照一定的模式，分配给左侧的变量。

## 语法

1. 数组解构：`let [a, b] = [1, 2];`
2. 对象解构：`let {x, y} = {x: 1, y: 2};`
3. 字符串解构：`let [a, b, c] = "abc";`
4. 数值解构：`let [a, b, c] = 123;`
5. 布尔值解构：`let [a, b, c] = true;`
   
```javascript
let [a, b] = [1, 2];
let {x, y} = {x: 1, y: 2};
let [a, b, c] = "abc";
let [a, b, c] = 123;
let [a, b, c] = true;
```
