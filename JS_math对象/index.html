Math 对象

Math 对象包含了一些常用的数学函数和常量。

    // 常量
  • Math.PI：圆周率，约等于 3.141592653589793。
  • Math.E：自然常数，约等于 2.718281828459045。
  • Math.SQRT2：平方根 2，约等于 1.4142135623730951。
  • Math.SQRT1_2：平方根 1/2，约等于 0.7071067811865476。
    • // 最大值和最小值
  • Math.max(10,20,30)：返回所有参数列表中的最大值: 30。
  • Math.min(10,20,30)：返回所有参数列表中的最小值: 10。
    • // 四舍五入
  • Math.ceil(3.1) // 4
  • Math.ceil(3.9) // 4
  • Math.ceil(-3.1) // -3
  • Math.ceil(-3.9) // -3
  • Math.floor(3.1) // 3
  • Math.floor(3.9) // 3
  • Math.floor(-3.1) // -4
  • Math.floor(-3.9) // -4
    • // 随机数
  • Math.random()：返回 0 到 1 之间的随机数。
  • Math.floor(Math.random() * 100) + 1：返回 1 到 100 之间的随机整数。
  • Math.abs(x)：返回 x 的绝对值。
  • Math.acos(x)：返回 x 的反余弦值。
  • Math.asin(x)：返回 x 的反正弦值。
  • Math.atan(x)：返回 x 的反正切值。
  • Math.atan2(y, x)：返回给定的 x 和 y 坐标值的反正切值。
  • Math.ceil(x)：返回大于或等于 x 的最小的整数。
  • Math.cos(x)：返回 x 的余弦值。
  • Math.exp(x)：返回 e 的 x 次幂。
  • Math.floor(x)：返回小于或等于 x 的最大的整数。
  • Math.log(x)：返回 x 的自然对数（底为 e）。
  • Math.max(x, y, z, ...)：返回参数列表中的最大值。
  • Math.min(x, y, z, ...)：返回参数列表中的最小值。
  • Math.pow(x, y)：返回 x 的 y 次幂。
  • Math.random()：返回 0 到 1 之间的随机数。
  • Math.round(x)：返回 x 四舍五入后的整数。
  • Math.sin(x)：返回 x 的正弦值。
  • Math.sqrt(x)：返回 x 的平方根。
  • Math.tan(x)：返回 x 的正切值。
